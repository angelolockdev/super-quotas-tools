# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-25


### Added
- Standardized premium SVG iconography for Anthropic, OpenAI, GitHub, Windsurf, Codeium, and Cursor.
- Staggered entrance animations for Dashboard and Platform Detail screens.
- GitHub Actions workflow for automated APK building and releases.
- Antigravity workflows for project setup and release management.
- Standardized project documentation and system manual structure.

### Changed
- Refactored `PlatformSummaryCard` and `PlatformDetailScreen` to use `flutter_svg`.
- Updated `AiPlatform` enum and brand colors for consistency.
- Standardized Supabase schema constraints for platform names.

### Fixed
- UI performance enhancements for 60fps interaction targets.
- Improved touch target sizes on dashboard cards.
